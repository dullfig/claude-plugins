# dullfig-plugins

Claude Code plugin marketplace by dullfig.

## Available plugins

| Plugin | Description | License |
|--------|-------------|---------|
| [memory-rlm](https://github.com/dullfig/memory  -rlm) | Persistent project memory for Claude Code | BUSL-1.1 |

## Install

```
/plugin marketplace add https://github.com/dullfig/claude-plugins
/plugin install claude-rlm
```
